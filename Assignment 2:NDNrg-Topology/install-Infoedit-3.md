**Installing Infoedit**

Infoedit is used to edit configuration files such as NFD configuration file.

To install infoedit:https://github.com/NDN-Routing/infoedit

<pre>
$ git clone — depth 1 https://github.com/NDN-Routing/infoedit
</pre>
